# WoWidget setup progress tracker update

This update replaces the tall vertical setup tracker with a compact horizontal navigation bar.

## Changed files

- `docs/assets/stylesheets/extra.css`
- all five files under `docs/getting-started/`

## Behavior

- Each stage is a working link to its setup page.
- Completed stages display a check mark.
- The current stage is highlighted with the accent color and bottom indicator.
- On narrow screens, the tracker stays horizontal and can scroll rather than becoming a tall vertical stack.
