<section class="about-story-grid">
  <article class="about-story-card">
    <span class="about-story-card__eyebrow">Why WoWidget exists</span>
    <h2>A character showcase built with more care.</h2>
    <p>WoWidget began as a personal project: a way to create a cleaner, more polished World of Warcraft character widget than the existing options allowed.</p>
    <p>Every part of the project—from the desktop application and portrait tools to the website and documentation—has been built independently with the goal of making character profiles feel more personal.</p>
  </article>

  <article class="about-story-card about-story-card--accent">
    <span class="about-story-card__eyebrow">Free and fan-built</span>
    <h2>No subscriptions. No premium tier.</h2>
    <p>WoWidget is completely free to use and is intended to remain that way. There are no paid features, subscriptions, advertisements, or intentionally locked functionality.</p>
    <p>This is a community project made by a World of Warcraft player, for other players who enjoy personalizing the way their characters are presented.</p>
  </article>
</section>
